# crf

TUI Based Filer Written in Crystal.

## Installation

```
$ git clone https://github.com/kazto/crf
$ cd crf && crystal build src/crf.cr
$ cp ./crf /path/to/installdir/
```

## Usage

```
  crf OPEN_PATH
```
